# Instagram-Engagement-Analysis

## Project Overview
This project focuses on identifying the key drivers of user engagement on Instagram. By analyzing a wide array of posting variables, the analysis aims to provide a data-backed strategy to maximize likes, comments, and overall reach. The goal is to provide actionable insights for content creators to improve their social media presence.

## Tech Stack & Tools
- **R**: Used for data cleaning, statistical modeling, and visualization.
- **Multiple Linear Regression**: Applied to quantify the impact of various posting features (e.g., time of day, caption length, hashtag count) on engagement.
- **Optimization Algorithms**: Utilized to determine the optimal posting schedule and content mix based on the regression results.

## Methodology
The analysis was performed using the following steps:
1. **Data Collection**: Engagement data was sourced from diverse Instagram accounts across different industries.
2. **Feature Engineering**: Created new variables such as 'time of day', 'day of week', and 'sentiment score' of captions.
3. **Model Building**: Developed a Multiple Linear Regression model to predict engagement levels.
4. **Optimization**: Used mathematical optimization to find the "sweet spot" for each variable to maximize the predicted engagement.
5. **Strategy Formulation**: Translated the model's numerical findings into a clear, non-technical posting strategy for stakeholders.
